# Инструмент авто-гаммы и вайбранса для Tarkov (ранее tarkov-settings)
![screenshot](./1.png)

**Language:** [English](README.md) | [한국어](README.ko.md) | Русский | [中文](README.zh.md) | [日本語](README.ja.md)

Это поддерживаемый форк [incheon-kim/tarkov-settings](https://github.com/incheon-kim/tarkov-settings), который не обновлялся с октября 2023 года и имеет несколько неисправленных отчётов о сбоях. В этом форке исправлено и добавлено:
- Сбои при потере валидности дескриптора дисплея NVIDIA (смена RDP-сессии, закрытие панели управления NVIDIA, засыпание/переподключение монитора) — см. issue [#3](https://github.com/incheon-kim/tarkov-settings/issues/3) и [#17](https://github.com/incheon-kim/tarkov-settings/issues/17) в оригинальном репозитории
- Настройки не сохранялись при закрытии окна кнопкой X в заголовке (раньше сохранение работало только через значок в трее → Exit)
- Сбой приложения при повреждённом или некорректном `tarkov-gamma-vibrance-tool.config.json` (теперь сбрасывается на значения по умолчанию с предупреждением)
- В список отслеживаемых процессов по умолчанию добавлен `EscapeFromTarkovArena` — см. issue [#23](https://github.com/incheon-kim/tarkov-settings/issues/23)
- Профили цвета: сохранение и загрузка нескольких именованных наборов Яркости/Контраста/Гаммы/Насыщенности — см. раздел [Профили](#профили) ниже
- Глобальные горячие клавиши для мгновенного переключения профилей, даже когда в фокусе другое окно — см. раздел [Горячие клавиши](#горячие-клавиши) ниже — реализует то, что запрашивалось в issue [#1](https://github.com/incheon-kim/tarkov-settings/issues/1) и [#12](https://github.com/incheon-kim/tarkov-settings/issues/12)

## [->**Скачать последнюю версию**<-](https://github.com/imda564/tarkov-gamma-vibrance-tool/releases/latest)

Автоматически меняет настройки цвета для [Escape from Tarkov](https://escapefromtarkov.com).

## Как это работает?
- Меняет значение Digital Vibrance через настройки NVIDIA с помощью [NvAPIWrapper](https://github.com/falahati/NvAPIWrapper)
- Меняет гамму с помощью некоторых [вызовов Win32 API](https://docs.microsoft.com/en-us/windows/win32/api/wingdi/nf-wingdi-setdevicegammaramp)

Цвета экрана меняются только тогда, когда окно Escape from Tarkov находится в фокусе.
Это обеспечивает плавный переход при сворачивании/разворачивании.

## Поддерживаемые видеокарты
- NVIDIA GPU **полностью поддерживается** (Яркость/Контраст/Гамма/Насыщенность)
- AMD GPU **поддерживается частично** (кроме насыщенности)
- **Intel/другие пока не поддерживаются.**

## Что умеет приложение?
Вы можете менять следующие настройки цвета:
1. Яркость (Brightness)
2. Контраст (Contrast)
3. Гамма (Gamma)
4. Digital Vibrance Control (он же Насыщенность)
5. Действует только когда окно EFT в фокусе (также предотвращает **резкую вспышку при переключении окон**)

## Как использовать
1. Откройте приложение (SmartScreen может предупредить, так как оно не подписано)
2. Установите нужные значения цвета
2.1. Двойной клик по подписи ползунка сбрасывает значение.
3. Сверните приложение и играйте в EFT
4. Закройте приложение, если хотите отключить эффект

**Настройки сохраняются в `tarkov-gamma-vibrance-tool.config.json` при закрытии окна или выходе через значок в трее.**

## Профили
Введите имя в поле в левом нижнем углу и используйте кнопки **Save** / **Load** / **Del**, чтобы сохранять и загружать несколько наборов значений яркости/контраста/гаммы/насыщенности (например, отдельный профиль для каждой игры или времени суток). Профили сохраняются в `tarkov-gamma-vibrance-tool.config.json` вместе с остальными настройками.

## Горячие клавиши
Сначала сохраните профиль, затем нажмите **Set** рядом с "Hotkey:" и введите сочетание клавиш (например, `Ctrl+F9`), чтобы привязать его к этому профилю. Горячая клавиша работает глобально — нажатие срабатывает, даже если в фокусе EFT (или любое другое окно), и сразу применяет яркость/контраст/гамму/насыщенность из этого профиля. Кнопка **Clear** снимает привязку. Если сочетание уже занято другим приложением, появится предупреждение — выберите другое.

## Предупреждение
1. Экран может пару раз мигнуть при активации окна EFT — это нормально, не переживайте.
2. **Отказ от ответственности: неизвестно, забанит ли BSG за использование этой программы.**
3. AMD поддерживает только Яркость/Контраст/Гамму.
4. Видеокарты Intel не поддерживаются.
5. Работает только в **режиме Borderless.**
6. Среда NVIDIA Optimus (в основном ноутбуки) не тестировалась.
